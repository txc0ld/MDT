# GlimmerTales Phase 0 Style Guide

Status: locked v0.1 for prototype and prompt pipeline.  
Rule: this guide describes visual attributes only. Do not reference, imitate, or name any existing studio, franchise, or protected character style in prompts, filenames, docs, or production notes.

## Product mood

GlimmerTales is a calm shared bedtime picture book: a parent reads aloud from warm, static illustrated pages while the child looks at friendly original characters. The art should feel tactile, cozy, rounded, premium, and low-arousal. It should never feel like a game, cartoon chase sequence, bright video, ad, or noisy app.

## Style Token Block, Daytime

Soft stylized 3D-rendered storybook image; rounded chunky tactile forms; handcrafted toy-render surface quality; matte-to-soft-specular materials; gentle fabric/clay-like micro texture; soft global illumination; warm diffuse key light; gentle ambient occlusion; no hard shadows; subtle subsurface scattering for a soft alive feel; friendly exaggerated proportions with large expressive eyes and generous head-to-body ratio; clean readable silhouettes; squash-and-stretch appeal in poses; warm saturated-but-not-garish palette; storybook depth of field with soft background bokeh; uncluttered composition; clear negative space reserved for app text overlay; cozy premium picture-book finish; no baked-in text anywhere.

## Style Token Block, Bedtime

Soft stylized 3D-rendered storybook image; rounded chunky tactile forms; handcrafted toy-render surface quality; matte-to-soft-specular materials; gentle fabric/clay-like micro texture; low-luminance bedtime palette of deep indigo, plum, moss, warm amber, and muted gold; soft candle-warm key light; gentle moonlit rim light but no blue-white glare; soft global illumination; gentle ambient occlusion; no hard shadows; subtle subsurface scattering for a soft alive feel; friendly exaggerated proportions with large expressive eyes and generous head-to-body ratio; clean readable silhouettes; calm poses; storybook depth of field with soft background bokeh; uncluttered composition; clear negative space reserved for app text overlay; cozy premium picture-book finish; no baked-in text anywhere.

## Negative Block

No text, no letters, no captions, no speech bubbles, no signs with words, no watermarks, no signatures, no logos, no readable symbols, no brand or franchise resemblance, no named-studio style, no photorealism, no harsh shadows, no blue-white glare, no scary faces, no uncanny faces, no sharp teeth emphasis, no threat, no extra limbs, no malformed fingers or antennae, no inconsistent eye style, no off-model proportions, no clutter over the reserved text zone, no autoplay/video/gamified visual language.

## Layout requirements for illustrations

- Default aspect ratio: 4:3 landscape for tablet-friendly read-together pages.
- Keep a calm text-safe zone on every page, preferably lower third or upper-left cloud/sky/meadow area. The zone must contain visual texture only, never baked-in words.
- Characters should not occupy the text-safe zone unless the story beat requires a close-up and the text zone moves.
- Avoid signs, boards, scrolls, speech bubbles, labels, title cards, and any prop that invites text generation.
- Facial emotion must be readable at tablet size from one meter away.
- Bedtime scenes should reduce saturation and contrast compared with daytime scenes.

## Reference input audit

The supplied images establish useful direction: rounded toy-like characters, warm jungle/meadow settings, expressive eyes, soft depth, rich foliage, and friendly humor. They also reveal the main risks to reject: baked-in signs, speech bubbles, captions, title banners, bright daytime blue skies for bedtime, and character drift across multi-panel scenes. Phase 0 therefore separates illustration from story text completely and reserves text for the app layer only.
